# Sorteador de Números Aleatórios

## Sobre o projeto

Este projeto consiste em um sistema de sorteio de números aleatórios desenvolvido com JavaScript, onde o usuário pode definir um intervalo numérico e a quantidade de números que deseja sortear.

A aplicação realiza validações para garantir que os dados informados sejam válidos, evitando intervalos incorretos e quantidades superiores à quantidade de números disponíveis no intervalo escolhido.

Após o sorteio, os números gerados são exibidos dinamicamente na tela, sem repetição entre os valores sorteados.

Além disso, o sistema também permite reiniciar completamente o sorteio, limpando os campos e resetando os resultados exibidos.

Neste projeto fui responsável pelo desenvolvimento completo da lógica da aplicação utilizando JavaScript.

A interface visual do sistema já estava previamente estruturada, permitindo foco total na implementação das funcionalidades, validações e regras do sorteio.

---

## Funcionalidades

- Sorteio de números aleatórios
- Definição de intervalo numérico
- Escolha da quantidade de números sorteados
- Validação de dados informados pelo usuário
- Bloqueio de números repetidos
- Exibição dinâmica dos resultados
- Reinício completo do sistema
- Alteração dinâmica de classes CSS
- Atualização interativa da interface

## Objetivo do projeto

O principal objetivo deste projeto foi praticar lógica de programação, manipulação de arrays, validação de dados e geração de números aleatórios utilizando JavaScript puro.
